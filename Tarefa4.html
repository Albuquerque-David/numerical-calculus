Exercício 1.1.2¶

Para este exerccio, utilizaremos a tcnica do Mtodo de Newton para calcular uma aproximao de ln(3) utilizando 20 passos do mtodo.

Para isto, queremos uma funo $f(x)$ tal que $f(ln(3)) = 0$. Como sabemos calcular $e^{x}$ para qualquer $x$ e sabemos as propriedades de $ln$, podemos utilizar a funo $f(x) = 3 - e^{x}$.

Com isto, temos que

$$f(x) = 3 - e^{x}$$$$f'(x) = -e^x$$

Assim, podemos utilizar o Mtodo de Newton na forma

$$x = x_{init} - \dfrac{f(x_{init})}{f'(x_{init})} = x_{init} - \dfrac{3-e^{x_{init}}}{-e^{x_{init}}}$$

tal que $x$ o nmero que queremos aproximar e $x_{init}$ um chute inicial.

Com isto, podemos utilizar o algoritmo abaixo para realizar a aproximao.

In [ ]:
# Exemplo de Metodo de Newton para achar ln(3)
function newton_ln_3(iteracoes, chute) # Nmero de iteraes e chute inicial
    for i=1:iteracoes
        chute = chute - (3-(Base.MathConstants.e^chute))/(-Base.MathConstants.e^chute) # Chute - f(chute)/f'(chute)
    end
    return chute # Aproximao do resultado. Quanto mais iteraes, maior a preciso.
end

Exercício 1.1.4¶

Para este exerccio, utilizaremos a tcnica da Interpolao Polinomial de Grau 1 para calcular uma aproximao de ln(3) e estimaremos o erro mximo. Utilizaremos ento um conjunto de pontos afim de montar o sistema de equaes que resultar na matriz de Vandermonde para encontrarmos uma funo aproximada que nos dar o valor de ln(3).

Podemos tirar uma aproximao da funo logaritmica prxima de um polinmio de grau 1 utilizando valores triviais, isto , $ln(1)$ e $ln(e)$, pois podem ser calculados usando apenas as propriedades de logaritmo. Com isto, teremos uma funo que passa pelos pontos $(x_{1},y_{1}),(x_{2},y_{2})$ e que nos dar um polinmio na forma $$P = c_{1}x + c_{0}$$

$$ \begin{cases} c_{1}x_{0} + c_{0} = y_{0} \\ c_{1}x_{1} + c_{0} = y_{1} \\ \end{cases} $$

Usaremos ento os pontos $(ln(1), 0),(ln(e),1)$. Portanto, temos que

\begin{cases} c_{1}ln(1) + c_{0} = 0 \\ c_{1}ln(e) + c_{0} = 1 \\ \end{cases}

Realizando as converses necessrias para exp, temos que, com isto, podemos modelar as matrizes da seguinte forma

$$ V = \begin{bmatrix} 1 & 1 \\ 1 & exp(1) \\ \end{bmatrix} $$$$ y = \begin{bmatrix} 0 \\ 1 \\ \end{bmatrix} $$$$ C = \begin{bmatrix} c_{0} \\ c_{1} \\ \end{bmatrix} $$

Devemos ento resolver o sistema tal que $V \cdot c = y$. Vamos ento utilizar o mtodo interpolacao_ln3_grau1 descrito abaixo para resolver o sistema, retornando os coeficientes do polinmio de grau 1.

In [ ]:
function interpolacao_ln3_grau1()
    # Cria a matriz V
    x = [1; exp(1)]
    y = [0; 1]
    V=[x.^0 x.^1]
    c=V\y # Resolve o sistema linear Vc=y
    return c #vetor de coeficientes
end

Podemos tirar uma aproximao da funo logaritmica prxima de um polinmio de grau 2 utilizando valores triviais, isto , $ln(1)$, $ln(e)$ e $ln(e^{2})$ pois podem ser calculados usando apenas as propriedades de logaritmo. Com isto, teremos uma funo que passa pelos pontos $(x_{1},y_{1}),(x_{2},y_{2}), (x_{3},y_{3})$ e que nos dar um polinmio na forma

$$P = c_{2}x^2 + c_{1}x + c_{0}$$$$ \begin{cases} c_{2}x_{0}^2 + c_{1}x_{0} + c_{0} = y_{0} \\ c_{2}x_{1}^2 + c_{1}x_{1} + c_{0} = y_{1} \\ c_{2}x_{2}^2 + c_{1}x_{2} + c_{0} = y_{2} \\ \end{cases} $$

\begin{cases} c_{2}ln(1)^2 + c_{1}ln(1) + c_{0} = 0 \\ c_{2}ln(e)^2 + c_{1}ln(e) + c_{0} = 1 \\ c_{2}ln(e^2)^2 + c_{1}ln(e^2) + c_{0} = 2 \\ \end{cases}

function vandermonde(x,y,grau)
    n,=size(y)
    V=zeros(n,grau+1)
    for i=1:n #linhas
        for j=1:(grau+1)
            V[i,j]=x[i]^(j-1)
        end
    end
    return V
end

function erro_total(x,y,modelo)
    n,=size(y)
    S=0
    for i=1:n
        S=S+(y[i]-modelo(x[i]))^2
    end
    return sqrt(S)  
end

Exercício 1.5)¶

Para este exerccio, devemos descobrir o horrio ou uma interpolao do horrio em que houve um assassinato. De acordo com a descrio, as temperaturas registradas foram

15h00: 34ºC¶

16h30: 30ºC¶

17h30: 25ºC¶

Seguindo a Lei do Resfriamento de Newton, teremos que a temperatura do corpo varia conforme:

$F(x) = ( Ti - Tf) e^{-kx} + Tf$

sendo a temperatura inicial $Ti = 37$, a temperatura final $Tf = 25$. Com isto, teremos que resolver o sistema no-linear para encontrar o coeficiente $k$ e determinarmos uma funo que encontre a hora do assassinato.

Podemos ento utilizar a funo calcula_assassinato em conjunto com as funes regresso e vandermonde para resolver o sistema no-linear fazendo a transformao do sistema utilizando $log$ e nos devolvendo os coeficientes da funo.

In [ ]:
function calcula_assassinato(x, y)
    x_barra=x
    y_barra=log.(y)
    V=vandermonde(x_barra,y_barra,1)
    coefs=regressao(x_barra,y_barra,1)
    return coefs
end

Com isto, podemos ento utilizar os pontos de tempo x temperatura medidos para montarmos o sistema e procurar os coeficientes que representam a funo esperada.

Com a funo que representa o assassinato, podemos ento utilizar o mtodo de Newton para nos auxiliar a encontrar o valor esperado (37C) na funo. Com isto, podemos utilizar a funo $f(x) = c1*e^{c2*x} - 37$ e sua derivada.

In [ ]:
function newton_assassinato(iteracoes, chute) # Nmero de iteraes e chute inicial
    for i=1:iteracoes
        chute = chute - (c1*(exp(c2*chute)) - 37)/(((c2*c1)*exp((c2*chute)))) # Chute - f(chute)/f'(chute)
    end
    return chute # Aproximao do resultado. Quanto mais iteraes, maior a preciso.
end
